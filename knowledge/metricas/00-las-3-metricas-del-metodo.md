# Las 3 Métricas del Método Audience (Pulados · Retención · Interacciones)

> *Clases 8-9 · Las únicas 3 métricas que importan para decidir si un reel funciona*

---

## La premisa

> Los algoritmos (IG · TT · YT) NO miran "seguidores" para amplificar.
> Miran **3 métricas específicas** en los primeros 30-120 minutos:
> 1. Pulados (skip rate)
> 2. Retención (watch time)
> 3. Interacciones (likes + comments + shares + saves)
>
> Si tu vídeo NO pasa el threshold de las 3 en su microgrupo inicial · nunca pasará a la siguiente ronda.

---

## MÉTRICA 1 · PULADOS (Skip Rate)

### Definición
% de personas que **saltaron antes del final** del vídeo / total de views.

### Fórmula
```
Pulados = (views - views_completas) / views × 100
```

### Benchmarks del método Audience

| Skip Rate | Estado | Significado |
|---|---|---|
| **>70%** | 🔴 CRÍTICO | Hook no funciona · headline débil · primer 1s mal |
| **50-70%** | 🟡 BAJO | Hook decente · pero no engancha |
| **<50%** | 🟢 BUENO | Hook funcionando · entras en zona competitiva |
| **<40%** | 🌟 POTENCIAL VIRAL | El algoritmo probablemente amplificará |
| **<30%** | 🚀 VIRAL CONFIRMADO | Caso Lili #21 estaba en 29.9% |

### Qué revisar si Skip Rate ALTO

1. **Primeros 0.5 segundos:** ¿pasa ALGO impactante visualmente?
2. **Primera palabra/sonido:** ¿activa atención?
3. **Texto en pantalla:** ¿se entiende en miniatura?
4. **Headline verbal:** ¿activa mínimo 2 gatillos?

### Cómo MEDIRLO en cada plataforma

- **Instagram:** Insights del reel → "Pulados" o "Skip rate"
- **TikTok:** Analytics → "Completion rate" (invertir: 100% - completion = skip)
- **YouTube Shorts:** Studio → "Swipe-up rate"

---

## MÉTRICA 2 · RETENCIÓN MEDIA (Watch Time)

### Definición
**Segundos vistos en promedio** / segundos totales del vídeo.

### Fórmula
```
Retención = avg_watch_time / video_duration × 100
```

### Benchmarks del método Audience

| Retención | Estado | Significado |
|---|---|---|
| **<15%** | 🔴 CRÍTICO | Mata el contenido · drop después del hook |
| **15-20%** | 🟡 BAJO | Hook OK · pero desarrollo aburre |
| **20-30%** | 🟢 ACEPTABLE | Mantiene atención · zona competitiva |
| **30-50%** | 🌟 BUENO | El algoritmo amplifica probable |
| **>50%** | 🚀 EXCELENTE | Viral garantizado si interacciones OK |
| **>70%** | 💥 VIRAL MASIVO | Algoritmo te empuja a top |

### Caso Lili #21
- **Watch time absoluto:** 36 segundos (en reel ~60s)
- **Retención:** ~60%
- **Resultado:** vídeo que destrabó su perfil · 100K seguidores en 11 días

### Qué revisar si Retención BAJA

1. **Pacing:** ¿cortes cada 2-4 segundos?
2. **Stacking de curiosidad:** ¿open loops · datos · misterios?
3. **Subtítulos:** ¿siempre presentes (85% ve sin sonido)?
4. **Pausas muertas:** ¿hay segundos sin nada?
5. **Repetición:** ¿dices lo mismo del hook varias veces?

### Cómo MEDIRLO

- **Instagram:** Insights → "Tiempo medio de visualización"
- **TikTok:** Analytics → "Total play time / Total plays = average watch time"
- **YouTube:** Studio → "Average view duration" + "Audience retention" (gráfico)

---

## MÉTRICA 3 · INTERACCIONES (Engagement)

### Definición
**(likes + comments + shares + saves) / views × 100**

### Fórmula
```
Engagement = (likes + comments + shares + saves) / views × 100
```

### Benchmarks del método Audience

| Engagement | Estado | Significado |
|---|---|---|
| **<3%** | 🔴 CRÍTICO | Contenido sin notable · no resuena |
| **3-5%** | 🟡 BAJO | Algo de conexión · pero débil |
| **5-10%** | 🟢 BUENO | Activa al avatar · base sólida |
| **10-15%** | 🌟 OBJETIVO | Alta probabilidad viral |
| **>15%** | 🚀 EXCELENTE | Algoritmo prioriza · escala rápida |
| **>20%** | 💥 BOMBA | Viral confirmado |

### Caso Lili #21
- 210K views
- **25K interacciones totales** = ~12% engagement
- **Resultado:** boost masivo del algoritmo

### Jerarquía de las interacciones (por valor)

1. **SHARE / Compartir** (señal máxima · activa moeda social)
2. **SAVE / Guardar** (segunda más fuerte · indica utilidad)
3. **COMMENT LARGO** (debate activo · señal alta)
4. **COMMENT corto** (señal media)
5. **LIKE** (señal mínima · ya no decide casi nada en 2026)

### Qué revisar si Engagement BAJO

1. **Notable:** ¿tienes 2-3 de los 8 elementos integrados?
2. **CTA:** ¿específico (no "dale like si te gustó")?
3. **Moeda social:** ¿compartirlo hace ver bien al avatar?
4. **Polarización:** ¿tomas posición fuerte (genera debate)?

### Cómo MEDIRLO

- **Instagram:** Insights → suma manual o Metricool
- **TikTok:** Analytics dashboard
- **YouTube:** Studio + Analytics

---

## El CUADRO DE DECISIÓN · 3 métricas combinadas

### Caso A · Las 3 BIEN
```
Skip <40% + Retención >35% + Engagement >10%
```
→ **VIRAL POTENCIAL** · ITERAR (genera 10 variantes con `iteracion-ganador`)

### Caso B · Skip ALTO + Retención BAJA
```
Skip >60% + Retención <20%
```
→ **HOOK FALLA** · revisar primeros 3 segundos · rehacer headline

### Caso C · Skip OK + Retención BAJA
```
Skip <50% + Retención <25%
```
→ **DESARROLLO FALLA** · pacing pobre · sin stacking · sin open loops

### Caso D · Retención OK + Engagement BAJO
```
Retención >30% + Engagement <5%
```
→ **NOTABLE FALTA** · técnicamente bien · pero no resuena emocionalmente · no se comparte

### Caso E · Skip MUY ALTO
```
Skip >70%
```
→ **DESCARTAR** · ni rehacer · idea/headline no conecta · pasar a siguiente idea

### Caso F · Skip OK + Retención OK + Engagement >15%
```
🚀 BOMBA
```
→ **REPLICAR INMEDIATAMENTE** · 10 variantes · diferentes ángulos · mismo insight

## La ventana de medición (timing crítico)

### Primeras 2 horas
- IG · TT · YT miden retención + skip
- Si pasas → ronda 2

### Primeras 24 horas
- Miden interacciones (saves + shares especialmente)
- Si pasas → ronda 3 (escalada masiva)

### Primeros 7 días
- Vida útil del contenido evergreen
- Para trending: se acaba en 3-5 días

### Importante
**NO juzgues el vídeo en las primeras 24h.** Espera 48-72h para tener data fiable.

---

## La regla de oro

> Si las 3 métricas están en zona "🟢 BUENO o superior" → REPLICAR
> Si 2 de 3 están bien → ITERAR (mejora la débil)
> Si solo 1 está bien → DESCARTAR (no merece más esfuerzo)

## Caso completo · Lili #21

| Métrica | Valor | Estado |
|---|---|---|
| Views | 210K | 🟢 |
| Skip rate | 29.9% | 🚀 VIRAL CONFIRMADO |
| Watch time | 36s (de ~60s · ~60%) | 💥 VIRAL MASIVO |
| Engagement | 25K / 210K = 12% | 🌟 OBJETIVO |

**Decisión correcta de Lili:** REPLICAR · generó múltiples variantes · de 0 a 100K seguidores en 11 días.
