# Formato 02 · RANKING ORDENADO CON ICONOS

> Lista numerada con iconos visuales · cada item revela mientras hablas. Perfecto para "Top N" educativo.

---

## Cuándo usar

- Contenido educativo con jerarquía clara (top 5 · top 8 · top 10)
- Tienes SISTEMA estructurado para enseñar (cada item = parte del sistema)
- Posicionarte como creador de método/framework
- Saves altísimos (gente guarda para revisar)

## Cuándo NO usar

- Sin diferencia real entre items (no hay ranking real)
- Audiencia ultra-corta (TT swipe rápido · no aguantan 8 items)

## Templates virales compatibles
- Template 1 · Ultra-específico (cada paso es ultra-específico)
- Template 2 · Autoridad (tu sistema = tu autoridad)
- Template 10 · Autoridad en X
- Template 12 · Transforma X en Y

## Estructura del guion

```
TEXTO OVERLAY INICIAL: RANKING [TEMA SISTEMA/CATEGORÍA]

[SEGUNDO 0-3] · HOOK (4 ganchos posibles)
[VISUAL]: Plano medio
[AUDIO]: gancho elegido
[OVERLAY]: gancho textual

[SEGUNDO 3-X] · ITEM 1
[ICONO]: 🧲 (emoji o sticker · representativo del item)
[AUDIO]: "[Categoría]. [Solución específica]. [Razón corta o expansión]."
[RESPUESTA QUE APARECE]: [SOLUCIÓN EN MAYÚSCULAS · LIMPIA]

[SEGUNDO X-Y] · ITEM 2
[ICONO]: 📢
[AUDIO]: "[Categoría]. [Solución]. [Razón breve]."
[RESPUESTA QUE APARECE]: [SOLUCIÓN]

[Continúa con 5-9 items · cada uno con icono distinto]

[ITEM FINAL] · CTA implícito
[ICONO]: 🤝
[AUDIO]: "¿Quién [hace todo esto]? Nosotros. Comenta [PALABRA] y lo miramos juntos."
[RESPUESTA QUE APARECE]: NOSOTROS

[TEXTO OVERLAY FINAL]: Comenta [PALABRA]
```

## Reglas críticas

1. **Cada item con icono ÚNICO** (no repetir · cada uno con emoji distinto)
2. **Estructura por item:** [Categoría] → [Tu solución específica] → [Razón corta]
3. **Pacing rápido:** 4-7 segundos por item máximo
4. **Item final = CTA disfrazado** ("¿quién monta esto? Nosotros")
5. **Solución en MAYÚSCULAS** en el overlay (más visible · memorable)

## Producción

- **Plano:** medio · sin movimiento (el visual es el ranking aparece)
- **Overlay:** template con casillas numeradas que se rellenan
- **Iconos:** emojis grandes o stickers prediseñados
- **Música:** rítmica · marca el cambio de item
- **Duración óptima:** 40-50 segundos
- **Tiempo grabación:** 20-30 min

## Métricas esperadas

- **Saves:** ALTÍSIMOS (info estructurada para revisar)
- **Watch time:** >60% (gente quiere ver todos los items)
- **Comments:** medios (preguntas sobre items específicos)
- **Conversión a seguidor:** muy alta (te perciben como sistema completo)

## Variaciones del formato

### Variación A · Ranking inverso (de peor a mejor)
Empieza con #10 · termina con #1 · sube tensión.

### Variación B · Ranking solo top 3
Más rápido · 30 segundos · alta retención.

### Variación C · Ranking comparativo
Cada item = comparación entre 2 opciones · más debate.

### Variación D · Ranking por categoría
"Top 3 de captación" · "Top 3 de venta" · "Top 3 de escala" · permite serie.

## Errores comunes

1. Iconos repetidos (resta originalidad)
2. Sin estructura clara [Categoría] → [Solución] → [Razón] (pareces divagar)
3. Pacing lento (>10s por item · audiencia se va)
4. CTA débil al final
5. Sin solución concreta · solo problemas (frustrante)
