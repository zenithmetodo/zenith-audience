## Tiempo de respuesta

- Issues críticos: <24h
- Issues normales: <7 días
- Feature requests: revisamos mensualmente

## Por favor NO

- Solicitar soporte por DM Twitter (usa email · es trackeable)
- Reportar bugs sin pasos para reproducir
- Pedir features sin describir el problema que resuelve

## Cómo ayudarnos a ayudarte

Al reportar bugs · incluye:
1. Versión Zenith Audience (`cat .claude-plugin/plugin.json`)
2. Versión Claude Code
3. OS (Mac · Windows · Linux)
4. Pasos exactos para reproducir
5. Logs si aplica
