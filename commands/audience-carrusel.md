# /audience-carrusel

Carrusel estructurado completo.

## Uso

```
/audience-carrusel "tu tema"
```

## Lo que hago

Invoco pipeline:
1. `asunto-detector`
2. `gatillo-master`
3. `template-selector` + `template-aplicador`
4. `carrusel-architect` (8-10 slides)
5. `notable-auditor`

## Output

HTML estético con:
- Slide por slide (texto · visual · open loop)
- Sugerencias de diseño/IA-imagen
- Copy del post · hashtags · timing
- Análisis · POR QUÉ funcionará

## Tiempo
~3 min

## Coste
~$0.12

## Cuándo invocar
- Contenido educativo profundo
- Listas largas (>5 puntos)
- Comparativas
- Para alta tasa de SAVES
