# /audience-reel

Guion completo de reel · todo en uno.

## Uso

```
/audience-reel "tu idea"
```

## Lo que hago

Invoco pipeline completo:
1. `asunto-detector` (valida la idea)
2. `gatillo-master` (aplica gatillos)
3. `template-selector` + `template-aplicador` (estructura)
4. `headline-3-tipos` (3 headlines)
5. `reel-architect` (guion completo)
6. `notable-auditor` (audita antes de entregar)

## Output

HTML estético PDF-ready en `output/reels/[fecha]-[título].html`:
- Setup (template · gatillos · categorías)
- 3 headlines (verbal · visual · audio)
- Guion completo palabra a palabra (hook + desarrollo + CTA)
- Producción (b-roll · música · iluminación · vestuario)
- Distribución (copy · hashtags · timing)
- Análisis · POR QUÉ funcionará + EN QUÉ se basa
- Checklist antes de grabar

## Tiempo
~3 min

## Coste
~$0.15

## Cuándo invocar
- Para CADA reel que vas a grabar
- En batch de grabación (prepara varios)
