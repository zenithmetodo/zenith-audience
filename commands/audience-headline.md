# /audience-headline

10 headlines de cada idea.

## Uso

```
/audience-headline "tu idea"
```

## Lo que hago

Invoco en orden:
1. `gatillo-master` (aplica gatillos)
2. `template-selector` (elige template óptimo)
3. `template-aplicador` (aplica fórmula)
4. `headline-3-tipos` (genera verbal + visual + audio)

## Output

```
10 headlines · cada uno con:
- Headline verbal (texto)
- Headline visual (descripción primer frame)
- Headline audio (primer 0.5s)
- Template usado
- Gatillos
- Score estimado
```

## Tiempo
~1-2 min

## Coste
~$0.08

## Cuándo invocar
- Antes de grabar reel/carrusel
- Para A/B testing de hooks
- Cuando tu hook actual no engancha
