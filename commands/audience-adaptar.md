# /audience-adaptar ⭐

El comando para adaptar ideas ganadoras a tu Verdadero Yo.

## Uso

```
/audience-adaptar https://instagram.com/.../reel/...
/audience-adaptar @competidor
/audience-adaptar "tema de un viral que vi"
```

## Lo que hago

Invoco `ideas-ganadoras-adapter` con pipeline completo:

1. **Búsqueda multi-fuente:** web · Apify · IA generativa
2. **Extracción "idea ganadora":** identifica el insight clave
3. **Adaptación al núcleo:** cruza con tu enemigo · promesa · bordones
4. **Output 3 variantes:** reel + carrusel + hilo · cada uno con análisis

## Output

HTML estético con:
- Análisis del viral original
- Idea ganadora extraída
- 3 adaptaciones a tu núcleo (reel · carrusel · hilo)
- POR QUÉ funcionará para ti
- EN QUÉ se basa (clases · estudios · casos)

## Tiempo
~3-5 min

## Coste
~$0.60

## Cuándo invocar
- Diario (rutina de inspiración)
- Cuando ves un viral interesante en redes
- Para conectar trend × tu mensaje sin perder voz
- Antes de planificar semana
