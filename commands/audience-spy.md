# /audience-spy

Análisis profundo de competidor.

## Uso

```
/audience-spy @handle
```

## Lo que hago

Invoco `spy-competidor` con:
- Apify para scraping (gratis con $5/mes)
- youtube-transcript-api o faster-whisper para audio
- Análisis Opus 4.7

## Output

Reporte HTML con:
- Top 5 contenidos
- Patrones ganadores (gatillos · estructuras · templates)
- Análisis notable (cuáles de los 8 elementos usa)
- **GAPS NO EXPLOTADOS** (oportunidades para ti)
- Recomendaciones adaptadas a TU núcleo

## Tiempo
~3-5 min

## Coste
~$0.60-0.80

## Cuándo invocar
- Semanal (1 competidor top)
- Al entrar a nicho nuevo
- Cuando tu contenido no crece
