# /audience-engenharia

Engenharia reversa profunda de un viral.

## Uso

```
/audience-engenharia https://instagram.com/.../reel/...
```

## Lo que hago

Invoco `engenharia-reversa` con:
- youtube-transcript-api (si YT) o yt-dlp + faster-whisper (IG/TT)
- Análisis Opus 4.7 profundo

## Output

HTML estético con:
- Transcripción completa timestamped
- CAPA 1 · HOOK (verbal + visual + audio + gatillos)
- CAPA 2 · DESARROLLO (estructura · pacing · open loops)
- CAPA 3 · NOTABLE (cuáles de 8 elementos)
- CAPA 4 · NÚCLEO (verdadero yo aplicado)
- CAPA 5 · RAZÓN DE VIRALIDAD
- RECETA REPLICABLE
- CÓMO APLICAR A TU NICHO

## Tiempo
~3-4 min

## Coste
~$0.40

## Cuándo invocar
- Post-mortem de TUS virales
- Aprender de virales ajenos (sin copiar)
- Mensual (top 3 virales del mes)
